# Contributing

Contributions are welcome if they make the skills more useful for real research workflows.

Good contributions:

- sharpen a skill workflow with concrete examples;
- add a missing quality gate for paper claims, experiments, or GitHub releases;
- add a small command recipe that composes existing skills;
- fix unclear language or broken links.

Please keep every skill:

- plain Markdown;
- easy to audit;
- honest about uncertainty;
- free of private credentials or unpublished paper content.
