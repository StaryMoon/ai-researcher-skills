# Command: Daily Research Brief

Use `paper-tutorial-briefing`, `figure-caption-reader`, and `github-research-maintainer`.

Inputs:

- list of papers or PDFs;
- local experiment logs;
- changed GitHub repositories;
- yesterday's briefing if available.

Output:

- three most important paper updates;
- two experiment decisions;
- one repository maintenance action;
- open questions and next commands.
